2014/0228Fri松村沙友理りんご生活始まった んよっ(o・・o)

こんばんごっ(o・・o)
さゆりんごっ(o・・o)
 
 
 
 
 
 
宣伝失礼します(・ω<)-☆
 
 
 
 
関西の皆様
 
弁当少女でます(*∩ω∩)
 
２５時２９分からです(*∩ω∩)
ゲストは
１９歳のシンガー
Ｓｅｒｅｎａさん♡
 
松村の高校時代の思い出も
語っています♡
是非みてね♡
 
 
 
 
あと、関東のみなさんは
 
NOGIBINGO２みてなぁ♡
初のロケでスキーに行きました♡
めっちゃ楽しかったんよ♡
冬の思い出♡♡
 
 
 
 
 
そして明日の２３時３０分から
NHK BSプレミアムで
乃木坂４６SHOWが放送されますヾ(＠⌒ー⌒＠)ノ
コントもあり
曲もフルで披露してるので
是非みてください♡
私も 高橋みなみさんと玲香が
どんな対談をしたのか
すごく気になりますヾ(＠⌒ー⌒＠)ノ
 
 
 
 
あと、雑誌の情報もヾ(＠⌒ー⌒＠)ノ
 
今日、
2月２８日発売の月刊エンタメに
ろってぃと大阪コンビで
グラビア載ってますヾ(＠⌒ー⌒＠)ノ
セクシーかつキュートな二人を
チェックしてやぁ♡
 
いつか
七瀬もいれて
３人で何か出来たらええなぁヾ(＠⌒ー⌒＠)ノ
 
 
 
 
 
 
最近は乃木坂Mailばかり
送っちゃってブログが
ちゃんと更新できてないです(´•̥ ω •̥` )
ほんまに申し訳ないです(´•̥ ω •̥` )
 
同時に沢山のことをするの
苦手なんかしら(´•̥ ω •̥` )
 
自分、不器用ですからー(´•̥ ω •̥` )！
 
 
でも！ちゃんと更新するので
もうちょいお待ちを～(´•̥ ω •̥` )
 
 
前のブログ
コメント沢山
嬉しかったんよ～(´•̥ ω •̥` )♡♡
 
 
最近、風邪ひいてて
やばいので短いですが許してください(´•̥ ω •̥` )
 
 
 
またね！！
