2014/0325Tue松村沙友理りんご相撲っ(o・・o)

こんばんGOーﾌﾞｰﾝ(((⊂(  ･ω･)⊃
さゆりんGOーﾌﾞｰﾝ(((⊂(  ･ω･)⊃
 
 
 
今日はＭＪの収録があったよーヾ(＠⌒ー⌒＠)ノ
気づいたら片想いは
ほんまにいい曲やね♡
歌う度に思います。と
前も書いた気がするが
やっぱり思いますヾ(＠⌒ー⌒＠)ノ
 
 
 
今日は質問返しやってみよーヾ(＠⌒ー⌒＠)ノ
 
 
 
Ｑ.ヾ(＠⌒ー⌒＠)ノの次に好きな
顔文字は？
 
A.(・ω<)-☆これかな(・ω<)-☆
でもこれ使うとイラっとするって
スタッフさんにいわれたことあるぜ(・ω<)-☆
 
 
 
 
Ｑ.好きなパン屋さんは？
 
A.最近、差し入れにパンが
多くて嬉しいのですヾ(＠⌒ー⌒＠)ノ
パンめっちゃ好き♡
クイニーアマンも大好きなんですが
モンタボーの北海道牛乳パンが
ほんまに好きです！！
一日で１００個は食べれるわ♡♡
モンタボーさんに行かれた際は
北海道牛乳パン買ってみてください♡♡
 
 
 
 
Ｑ.前回のブログのお洋服どこの？
 
A.あれは
earth music&ecologyさんの
グリーンのワンピースです♡♡
大人っぽいシンプルなワンピースなので
大きめのネックレス付けました♡
お揃いが欲しいと言ってくださる方がいたのですが
結構前に買ったやつやから
もうないかも(´•̥ ω •̥` )
申し訳ない(´•̥ ω •̥` )
 
 
 
Ｑ.幸せマンボーって、なに？
A.マンボーっているやん♡♡
マンボーは幸せな顔してるから
幸せマンボー♡なの♡♡♡
ってか松村の言葉に意味を求めたらあかんヾ(＠⌒ー⌒＠)ノ
これ、りんご王国の常識よヾ(＠⌒ー⌒＠)ノ
 
 
 
Ｑ.ラブライブ！では
誰が好き？
 
A.ことりちゃんーー♡♡♡♡♡
にこちゃんも好きだが
やっぱりことりちゃんーーー♡
でも三森すずこさんが大好きやから
海未ちゃんも好きだ。
みもりん好きだ(´•̥ ω •̥` )
可愛すぎてはうあ～♡
みもりん♡アルバム買います～♡
ツアーも行きたい行きたい行きたい(´•̥ ω •̥` )
 
 
 
 
コメントに沢山
メールの返信書いてくれてて
めっちゃ嬉しい♡♡
ちゃんと届いてるって実感するねヾ(＠⌒ー⌒＠)ノ
ありがとう～♡
 
 
 
大相撲の春場所が終わりましたね～♡
 
鶴竜関の横綱への推薦決定に
すごく興奮しております(´•̥ ω •̥` )
白鵬関との戦いで
鶴竜関の気迫がすごくて
攻めて攻めてってしてるのが
ほんまにかっこ良くて素敵で
初優勝してほしい>_<！って
ずっと一日中思ってたから
ほんまに感激(´•̥ ω •̥` )！！
あついぜ！！！
もえるぜ！！！
興奮したぜ！！！
遠藤関も松村は凄いと思うよ！
負け越しちゃったけど
あれだけできたのも
凄いよ！！と
全くど素人の松村が言ってみる⁽˙³˙⁾◟( ˘•ω•˘ )◞⁽˙³˙⁾
でも同年代の方が頑張ってるのみて
松村もやらなあかん！って思わされます！！
とにかく楽しかった♡♡
生でみたい！！！！！
次こそは見に行きたい！！
 
 
 
 
日村さんとお相撲したら
勝てるかなぁ～♡？
尻相撲やったら勝てそうやなヾ(＠⌒ー⌒＠)ノ♡
